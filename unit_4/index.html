No more broken Urls, Have it your way.

It works on any device, and in any browser without needing any extensions or plugins. Your link pages are stored as PDF files. A new link will be generated for the PDF files which you can save directly in the documents so you can have access to the page inforation even if the page is removed without getting page not found error (404 error).
